A sleek, responsive single-page web application built to showcase and manage registrations for **The Nexus 2026** fest. 

The portal features a modern dark-mode UI with vibrant orange accents (`#FF9800`) and categorizes the fest into two main tracks: **Compile & Conquer** (Technical Challenges) and **TalentTornado** (Creative Arts & Expo).

## ✨ Key Features

* **Single-Page Architecture (SPA):** Smooth, instant transitions between the Home, Events, Details, and Registration sections without any page reloads.
* **Dynamic Event Rendering:** Event lists, details, rules, and contact information are generated dynamically using JavaScript arrays.
* **Modern UI/UX:** A visually striking dark theme styled with smooth hover animations, CSS grid layouts, and FontAwesome icons.
* **Fully Responsive:** Optimized for both desktop and mobile viewing.
* **Integrated Registration:** A functional frontend form that automatically captures the selected event and simulates the registration process.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure and form handling.
* **CSS3:** Custom variables, keyframe animations, Grid/Flexbox layouts, and responsive media queries.
* **Vanilla JavaScript (ES6):** State management, DOM manipulation, dynamic HTML generation, and event handling.
* **FontAwesome:** Scalable vector icons for the UI.
